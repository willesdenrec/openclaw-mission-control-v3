## 🚀 Getting Started

### **Local Development**
```bash
# Clone the repository
git clone https://github.com/willesdenrec/openclaw-mission-control-v3.git

# Navigate to project directory
cd openclaw-mission-control-v3

# Open in browser (no build process required)
open index.html
```

The dashboard is designed for static hosting and works perfectly with:
- **GitHub Pages** (automatically configured)

## 🎮 Usage Guide

### **Dashboard Overview**
1. **Top Header**: System status, time, and quick controls
2. **Left Panel**: Agent operations center with animated agents
3. **Center Top**: System dashboard with live metrics
4. **Right Top**: Activity stream with real-time events
5. **Center Bottom**: Task queue visualization
6. **Right Bottom**: Command center terminal
7. **Bottom Footer**: Connection status and system controls

### **Interactive Features**
- **Click Agents**: View detailed agent information and controls
- **Type Commands**: Control the system via terminal
- **Control Queue**: Pause/resume/clear task flow
- **Use Keyboard Shortcuts**: Power user controls
- **Toggle Fullscreen**: Immersive mission control experience

### **Keyboard Shortcuts**
- `Ctrl + /` - Show keyboard shortcuts help
- `Esc` - Close all modals and dialogs
- `Ctrl + F` - Toggle fullscreen mode
- `Ctrl + K` - Focus command field
- `Ctrl + Q` - Pause/resume task queue
- `Ctrl + C` - Clear task queue

### **Available Commands**
- `status` - Show system status and metrics
- `agent list` - List all active agents
- `deploy` - Initiate deployment sequence
- `monitor` - Show detailed monitoring data
- `help` - Show available commands
- `clear` - Clear command history
